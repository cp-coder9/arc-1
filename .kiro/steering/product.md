# Product Summary

Architex is a Built Environment OS — a role-based, action-driven platform coordinating the complete lifecycle of construction and architectural projects in South Africa.

## What It Does

- Guides projects through an 8-stage lifecycle: Brief → Appoint → Design → Comply → Procure → Build → Pay → Close-out
- Serves 17 user roles (clients, architects, engineers, contractors, quantity surveyors, etc.)
- Provides AI-powered compliance checking against South African SANS standards (10400-K, N, T, C, XA, etc.)
- Manages financial workflows (escrow, milestone payments, FICA reporting)
- Handles document control, drawing intelligence, procurement, and construction site execution
- Includes a CPD (Continuing Professional Development) assessment platform for professionals

## Key Characteristics

- Multi-tenant: firms and projects are scoped per user/organization
- Role-aware: navigation, dashboards, and available tools vary by user role
- Regulation-heavy: deep integration with South African building codes and professional body requirements
- AI-assisted: multi-agent system for compliance checking, matching, tender evaluation, and workflow recommendations
- Demo mode available via `VITE_DEMO_MODE=true` with per-user sandboxed data
