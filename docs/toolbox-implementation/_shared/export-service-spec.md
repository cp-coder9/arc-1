# Export Service Specification

## Current State

`StandaloneToolRunService.markExported()` only sets `exportedAt` and `exportFormat` — no actual file generation.

## Required: ExportService

```typescript
// src/services/tools/exportService.ts

export class ExportService {
  toPdf(run: StandaloneToolRun): Blob
  toCsv(run: StandaloneToolRun): Blob
  toJson(run: StandaloneToolRun): Blob
  download(blob: Blob, filename: string): void
}
```

### PDF Export

Generate a formatted document with:
- Header: tool label, date, run ID
- Input section: field name → value table
- Output section: structured results with formatting per type
- Footer: generated by Architex, not a professional document

Use browser `window.print()` or a PDF library. Start with `print()` for MVP:

```typescript
exportToPdf(run: StandaloneToolRun) {
  const printWindow = window.open('', '_blank')
  printWindow.document.write(`
    <html><head><title>${run.toolLabel}</title>
    <style>body{font-family:sans-serif;padding:2em}
    h1{font-size:1.5em} table{width:100%;border-collapse:collapse}
    td,th{border:1px solid #ccc;padding:8px;text-align:left}
    </style></head><body>
    <h1>${run.toolLabel}</h1>
    <p>Run ID: ${run.runId} | ${new Date(run.createdAt).toLocaleDateString()}</p>
    <h2>Input</h2>
    <table>${Object.entries(run.input).map(([k,v]) =>
      `<tr><td>${k}</td><td>${v}</td></tr>`).join('')}</table>
    <h2>Output</h2>
    <table>${Object.entries(run.output).map(([k,v]) =>
      `<tr><td>${k}</td><td>${typeof v === 'object' ? JSON.stringify(v) : v}</td></tr>`).join('')}</table>
    <p style="margin-top:2em;font-size:0.8em;color:#666">Generated by Architex</p>
    </body></html>
  `)
  printWindow.document.close()
  printWindow.print()
}
```

### CSV Export

Flatten output to rows. For table outputs (BoQ items), create multi-row CSV:

```typescript
toCsv(run: StandaloneToolRun): Blob {
  const rows = [['Field', 'Value']]
  for (const [key, value] of Object.entries(run.output)) {
    if (Array.isArray(value)) {
      value.forEach((item, i) => rows.push([`${key}[${i}]`, JSON.stringify(item)]))
    } else {
      rows.push([key, String(value)])
    }
  }
  const csv = rows.map(r => r.map(c => `"${c}"`).join(',')).join('\n')
  return new Blob([csv], { type: 'text/csv' })
}
```

### JSON Export

Full raw dump:

```typescript
toJson(run: StandaloneToolRun): Blob {
  return new Blob([JSON.stringify({ input: run.input, output: run.output, runId: run.runId, createdAt: run.createdAt }, null, 2)], 
    { type: 'application/json' })
}
```
