# Site Diary Standalone Entry — site_diary_entry

## Overview

| Field | Value |
|-------|-------|
| **ID** | `site_diary_entry` |
| **Category** | `site_management` |
| **Roles** | contractor, subcontractor, site_manager |
| **Current State** | **WORKING** — dedicated form (date, weather, description, labour, plant, deliveries) |
| **Priority** | P1 |
| **Branch** | `toolbox/site-diary-entry` |

## PRD

A site manager needs to record a daily site diary entry: weather, labour count, plant on site, deliveries, activities, delays, observations. Standalone, exportable to PDF for site records.
